# Tacoma EV Dashboard

**Project Description:**  
This project is an interactive dashboard for Tacoma and Washington State electric vehicle (EV) adoption and charging station analytics. It combines local and global EV data, displays trends via Tableau dashboards, and includes a Smartsheet-based collaboration hub.  

**Tech Stack:**  
- Backend: Node.js + Express  
- Frontend: React  
- Data: Python (pandas) for cleaning + CSV export  
- Visualizations: Tableau  
- Collaboration: Smartsheet  
- Hosting: Free tiers (Vercel/Netlify/Firebase)  

**Project Structure:**  
- /backend → Node.js / Express API
- /frontend → React dashboard
- /data → raw + cleaned CSVs
- /notebooks → Python Jupyter notebooks
- /reports → PDFs & insights
